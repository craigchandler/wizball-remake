All release packages are published on the [GitHub releases page](https://github.com/craigchandler/wizball-remake/releases).

## Desktop (Linux x86_64)

A self-contained tar.gz with the binary and all game data. Requires SDL2 to be installed.

[All releases →](https://github.com/craigchandler/wizball-remake/releases)

## Desktop (Windows x86_64)

A self-contained zip with the binary, SDL2 DLLs, and all game data.

[All releases →](https://github.com/craigchandler/wizball-remake/releases)

## PortMaster (aarch64 + armhf)

A combined `WizBall.zip` containing both architecture binaries for installation via [PortMaster](https://portmaster.games).

[Download latest WizBall.zip](https://github.com/craigchandler/wizball-remake/releases/latest/download/WizBall.zip)
